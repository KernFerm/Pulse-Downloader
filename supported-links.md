# Supported Video Sites

This application supports downloading from a wide range of video sites using two methods:

---

## 1. Direct File Download
Works for any site that provides a direct link to a video file (e.g., ending in `.mp4`, `.mkv`, `.webm`, etc.).

**Examples:**
- Public file servers
- Direct video links from blogs or news sites
- Educational platforms with open video links

---

## 2. yt-dlp Fallback (Advanced)
Supports hundreds of video sites via [yt-dlp](https://github.com/yt-dlp/yt-dlp).

**Popular Supported Sites:**
- YouTube
- Vimeo
- Dailymotion
- Facebook (public videos)
- Twitter/X (public videos)
- TikTok
- SoundCloud
- Instagram
- Reddit
- Twitch
- Bilibili
- PeerTube
- VK
- Many more!

For the full list of supported sites, see the official yt-dlp documentation:
- [yt-dlp Supported Sites](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md)

## ***NOTE*** 
> - you can not download from Netflix or any platforms like that ***DRM Protection***.

---

**Note:**
- Only download content you have permission to download.
- Some sites may require login or additional configuration for private/protected videos.
